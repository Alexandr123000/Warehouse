# Warehouse
## Description
The warehouse is designed for wholesale purchasing of goods and sales. It has certain functions, such as:
- purchasing goods
- selling goods
- displaying goods of certain type
- displaying all sold goods
- clearing all products in the warehouse
- sorting goods by name
- sorting goods by price
- grouping goods by type

Interaction with data is carried out through an SQLite database.
___

- WPF
